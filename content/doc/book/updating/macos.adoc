== Updating Jenkins on MacOS 

NOTE: MacOS is not supported by Jenkins, and the Homebrew package manager is maintained outside the Jenkins project.
The information on this page is based on

* link:https://docs.brew.sh/FAQ#how-do-i-update-my-local-packages[Homebrew Documentation]

for more information visit 

* link:/download/lts/macos[Installing Jenkins LTS on macOS]
* link:/download/weekly/macos[Installing Jenkins Weekly on macOS]

include::doc/book/updating/_pre-upgrade-wizard.adoc[]

To upgrade the Jenkins LTS on macOS, run the following commands in a terminal window with homebrew:
[source]
```
brew update && brew upgrade jenkins-lts
```
Similarly, to upgrade the Jenkins Weekly on macOS, run the following commands in the terminal:
[source]
```
brew update && brew upgrade jenkins
```
