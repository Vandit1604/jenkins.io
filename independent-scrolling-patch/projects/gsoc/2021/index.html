Jenkins in Google Summer of Code 2021

The Jenkins project participated in Google Summer of Code 2021. This page contains an archive of the 2021-specific information. See the main GSoC page for the actual information.

Projects

All five projects have been successfully completed in 2021. You can find the project information on the project pages:

  • CloudEvents Plugin for Jenkins by Shruti Chaturvedi

  • Conventional Commits Plugin for Jenkins by Aditya Srivastava

  • Git credentials binding for sh, bat, and powershell by Harshit Chopra

  • Jenkins Remoting Monitoring by Akihiro Kiuchi

  • Security Validator for Jenkins Kubernetes Operator by Pulkit Sharma
