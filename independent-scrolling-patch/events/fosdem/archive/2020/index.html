Jenkins at FOSDEM 2020

Jenkins project has participated in FOSDEM 2020. This event is coordinated by the Jenkins Advocacy and Outreach SIG. Current plan:

  • Jan 30 - Jenkins Pipeline Fundamentals training led by Mark Waite (Register here)

  • Jan 31 - Jenkins Contributor Summit (RSVP here)

    • This is a small event for active contributors and those who are interested in working on foundation projects: key architecture changes (UX, JCasC, Cloud native Jenkins, etc.), governance and infrastructure

  • Feb 01-02 - Conference dates (no registration needed)

    • We will have a Jenkins stand during the entire conference

    • On Sunday there will be a Continuous Integration and Continuous Deployment devroom led by Olivier Vernin. There will be talks about Jenkins there

    • See the full agenda here
