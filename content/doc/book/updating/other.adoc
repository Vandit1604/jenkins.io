== FreeBSD

When Jenkins is installed from the FreeBSD `pkg` system, it should be upgraded with `pkg`.
Connect to your FreeBSD machine, get root access and run the commands:
```
pkg update
```
```
pkg upgrade jenkins
```
```
service jenkins restart
```
