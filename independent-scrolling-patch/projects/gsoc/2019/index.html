Jenkins in Google Summer of Code 2019

Jenkins project participated in Google Summer of Code 2019. This page contains archive of the 2019-specific information. See the main GSoC page for the actual information.

Projects

Completed projects:

  • Multi-branch Pipeline support for Gitlab

  • Plugins Installation Manager CLI Tool/Library

  • Remoting over Apache Kafka with Kubernetes features

  • Role Strategy Plugin Performance Improvement

  • Working Hours Plugin - UI Improvements

Cancelled projects:

  • Artifact Promotion plugin for Jenkins Pipeline

  • Cloud Features for External Workspace Manager Plugin
