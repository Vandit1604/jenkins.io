Google Summer of Code

Overview

This special interest group organizes Google Summer of Code program within the Jenkins organization. The scope of work includes coordinating GSoC activities and working with potentials mentors and contributors in order to form GSoC project teams.

Description

This SIG focuses on Google Summer of Code and other similar events. The group drives the GSoC sub-project within the Jenkins organization. The scope of work includes organizing the events and working with potentials mentors and contributors in order to organize GSoC teams.

You can find more info about GSoC in Jenkins here.

Meetings

  • GSoC SIG topics are discussed during the Advocacy & Outreach topics. See its information page for further details.

  • See the GSoC sub-project for the list of public events within Google Summer of Code time frame

  • Other meetings will be announced in GSoC mailing lists
